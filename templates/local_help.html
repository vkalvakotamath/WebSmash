Local Comparator Help | WebSmash


If you are using the local comparator, run the WebSmash.py script from Github or run your own script to fetch pictures and corresponding metadata into the Profiles folder. To do this, clone the Github repository locally, remove all Cat pictures provided as examples, and change them with the web-data you scraped either with the WebSmash.py script or your own script. You should then be able to run the local comparator.
